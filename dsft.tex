\maketitle
This package provides a doublestroke version of the character $1$ and other characters added after.

\medskip

\commandline{dsone}This command produces $\dsone$.
\commandline[\argt{sub}]{ind}This command outputs the indicator function of $\emph{sub}$ and produces for instance $\ind E$.
\commandline{oldsquare}This command produces $\oldsquare$. It corresponds to the \texttt{\textbackslash square} command provided by the \ensuremath{\mathcal{A}\kern-.1667em\lower.5ex\hbox{\ensuremath{\mathcal{M}}}\kern-.125em\mathcal{S}}.
\commandline{oldstar}This command produces $\oldstar$. It corresponds to the \texttt{\textbackslash star} command provided by \LaTeX.
\commandline{oldtriangle}This command produces $\oldtriangle$. It corresponds to the \texttt{\textbackslash triangle} command provided by \LaTeX.
\commandline{star}This command produces $\star$.
\commandline{square}This command produces $\square$.
\commandline{triangle}This command produces $\triangle$.
\commandline[\argt{level}]{filledstar}This command gradually fills the star provided by the \texttt{\textbackslash star} command. The fill level goes from 0 to 5 (included). Any \emph{level} greater than 5 will be with display a hbox, with width the one of the \texttt{\textbackslash star} glyph.

The different glyphs (produced as \texttt{\textbackslash Large}) are: {\Large$\filledstar0\filledstar1\filledstar2\filledstar3\filledstar4\filledstar5$}.
